# Emplace

Gem for cmake build settings

## Installation

Add this line to your application's Gemfile:

    gem 'emplace'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install emplace

## Usage

	require 'emplace'
	Emplace.cmake
	Emplace.build
	Emplace.package('package-name')
